# 🚢 Titanic Survival Prediction using Machine Learning

## 📌 Overview

This project focuses on predicting whether a passenger survived the Titanic disaster using machine learning techniques. It is one of the most popular beginner datasets used to understand real-world classification problems.

The goal is to analyze passenger data and build a predictive model that can estimate survival outcomes based on different features.

---

## 🎯 Problem Statement

The sinking of the RMS Titanic in 1912 was one of the deadliest maritime disasters, with over 1,500 lives lost. ([GeeksforGeeks][1])

The objective of this project is to:

* Build a classification model
* Predict whether a passenger survived or not
* Identify key factors that influenced survival

---

## 📊 Dataset

This project uses the famous **Titanic dataset from Kaggle**, widely used in machine learning.

The dataset contains passenger details such as:

* Age
* Gender
* Passenger Class (Pclass)
* Fare
* Number of siblings/spouses (SibSp)
* Number of parents/children (Parch)
* Embarked location

The target variable:

* `0` → Did not survive
* `1` → Survived

This dataset is commonly used for supervised learning tasks where the goal is to predict survival outcomes based on passenger characteristics. ([Hugging Face][2])
